Linux Advanced Commands

1. vim

This is an enhanced version of VI text editor. In Linux it is used to add and edit text.

   vim filename

2. chown

It is used to change the owner or group of a file or directory. And it gives all read, write and execute permission to the corresponding owner.

   chown newowner filename

3. chmod

The chmod command is used to allow an administrator to set or modify a file’s permissions. If you are giving a 777 permission to user, then they get permission to read, write and execute the file..

a). Changing Permissions Using Octal Notation:

for giving read and write permission to user, read permission to group and others.

   chmod 644 filename

b). Changing Permissions Using Symbolic Notation:

for giving execute(x) permission to users. 

  chmod u+x filename

c). Removing Execute Permission:

his command removes execute permission for all (a) users from the script file 

  chmod a-x script.sh

4. chgrp

It is used to change the group ownership of a file or directory. And it gives all read, write and execute permission to the corresponding owner.

   chgrp newgroup filename

5.grep

It works as a filter that searches for lines matching a specific pattern and prints the matching lines to standard output. This command searches for the specified string ("search_string") in the file filename and prints the lines containing the match.

   grep "search_string" filename

6. scp

SCP (Secure Copy Protocol) is a network protocol that securely copies files/folders between Linux (Unix) systems on a network. It is an alternate command of cp

   scp username@remote_host:/path/to/source/file.txt /local/destination/

7. ssh

Secure Shell (SSH) is a widely used network protocol that provides a secure way to access remote servers and computers.

  ssh username@remote_host

8. ping

It is a utility that helps to test connectivity between two devices on a network. ping command sends a request to a specified device and waits for a response.

  ping example.com

9. telent

the telnet command is used to create a remote connection with a system over a TCP/IP network. It allows us to administrate other systems by the terminal.

  telnet example.com

10. nc

It stands for NetCat. It is a powerful tool for network diagnostics and data transfer. NC can operate in two modes: listen mode and connect mode. NC can be used for file transfers, port scanning, banner grabbing, and even setting up a chat system or a reverse shell. It is a more effective way than telnet to check if a specific port is open on a destination system. 

   nc example.com 80  
  
This command connects to the host example.com on port 80.

11. yum

Yum is a command-line package manager for RPM-based Linux systems, such as Red Hat Enterprise Linux, CentOS, Fedora, and Oracle Linux. It stands for "Yellowdog Updater, Modified" 

   sudo yum install package_name

12. dnf

dnf command is a package manager command in Red - based Linux distros like Centos, fedora and RHEL. It stands for Danified YUM.

  sudo dnf install package_name

13. rpm

RPM is a command-line utility for managing packages on Unix/Linux systems. It allows you to install, query, update, verify and remove RPM packages.

It is the default package manager for Red Hat based systems and only works with the .rpm format. 

   sudo rpm -ivh package.rpm

14. df

The df command displays information about total space and available space on a file system. 

   df /dev/sda1

15. tar

tar stands for “tape archive”. tar, by default, keeps the directory structure of archived files. It is used to take backup and restore data in files.

-cvf: create verbose forcefully

-xvf: extract verbose forcefully 

   tar -cvf archive.tar files_directory

-c: Create a new archive.

-c: Create a new archive.

-f: Specify the archive file name. 

tar -xvf archive.tar

-x: Extract files from an archive.

-v: Verbosely list the files processed.

-f: Specify the archive file name.

16. zip

A zip command is a powerful file packaging utility that can be used to compress and extract files and directories, as well as to add, update, and delete files from a zip archive

   zip archive.zip file1.txt file2.txt directory/

17. gzip

Gzip command is used to compress the size of files or directories. When we apply the gzip command, then the .gz extension. Gunzip command is used to decompress the compressed size of the file to its original size. After gunzip, the .gz extension gets removed by itself.

   gzip file.txt

18. ps

The ps command in Unix-like operating systems, including Linux, is used to display information about currently running processes. It provides a snapshot of the processes running on the system at the moment the command is executed. The basic syntax of the ps command is:

   ps [OPTIONS]

19. awk

awk is used for manipulating data and generating report. It requires no compiling and allows the user to use variables, numeric functions, string functions and logical operators.

   awk 'pattern { action }' input_file

20. sed

It stands for stream editor and it can perform lots of functions on file like searching, find and replace, insert and delete.

   sed [OPTIONS] 'COMMAND' input_file

21. wget

Wget is a networking command-line tool that lets you download files and interact with REST APIs. It supports the HTTP , HTTPS , FTP , and FTPS internet protocols. 

   wget [OPTIONS] [URL]
    wget https://example.com/file.zip

22. curl

It is a command line tool that enables data transfer over various network protocols. It communicates with a web or application server by specifying a relevant URL and the data that need to be sent or received.

   curl [OPTIONS] [URL]

23. elinks

ELinks is a free text-based web browser for Unix-like operating systems

   elinks https://example.com

24. free

It allows you to check for memory RAM on your system or to check the memory statics of the Linux operating system.

  free [OPTIONS]

25. top

The top (table of processes) command shows a real-time view of running processes in Linux and displays kernel-managed tasks. The command also provides a system information summary that shows resource utilization, including CPU and memory usage.

26. netstat

The netstat command generates displays that show network status and protocol statistics.

  netstat [OPTIONS]

27. dmidecode

It is a command- line tool on Linux systems that provides information about the system’s hardware components.

   dmidecode [OPTIONS]

28. traceroute

traceroute in Linux is a command which enables the system to map the journey of the packet which contains information from source to destination.

This method is very helpful in modern times in order to keep a track of the millions and millions of data transacted at each second.

   touch filename.txt

29. mtr

It stands for My TraceRoute. The path your packets are taking (including the IP addresses of each router) .

   touch filename.txt

30. tcpdump

It is a packet analyzer that is launched from the command line. It can be used to analyze network traffic by intercepting and displaying packets that are being created or received by the computer it's running on. 

   tcpdump [OPTIONS] [FILTER_EXPRESSION]

31. nslookup

Use this command to find the address record for a domain. It queries domain name servers and gets the details.

   nslookup example.com 8.8.8.8
