Linux Basic Commands

1. whoami

The whoami command is used to display the effective user ID of the current user.

It prints the username associated with the current user's session.

2. ls

It is used to list down the names and features of files and directories.

   ls

3. ll

It is used to list the long information about files and directories in the system.

   ls -l
    ll

4. cal

The cal command in Linux is used to display a calendar in the terminal. By default, it shows the current month's calendar.

5. date

The date command in Linux is used to display the current date and time or to set the system date and time. Here's an overview of the date command:

6. pwd

The pwd command in Linux stands for "Print Working Directory." It is used to display the current working directory, which is the directory you are currently located in within the file system.

7.touch

It is used to create, modify and change a blank file.

   touch filename.txt

8. mkdir

It is used to make a directory(in Linux) or folder(in windows)

   mkdir dirname

9. cat

It is used to see the data added in a file. When we use “cat (file name)”, then it adds data in the file. 

   cat filename

a). Create a New File or Overwrite Existing File:

  cat > newfile

b). Append Contents to an Existing File:

cat >> existingfile_name

10.head

It is used to filter the head lines to a particular number of lines (for e.g. – if we want to see data of starting 4 lines, then we use: head –n 20 (file name).

   head -n 20 filename

11. tail

It is used to filter the tail lines to a particular number of lines (for e.g. – if we want to see data of starting 4 lines, then we use: tail –n 20 (file name)

   tail -n 20 filename

12. less

This command is used to read the contents of a text file’s one page (one screen) at a time.

   less filename

13. more

This command is used to read the contents of a text file’s one page. At the last page there is a “more” option which appends the data up to the next page.

   more filename

14. echo

Echo command is used to print the text or strings which are passed as arguments on the command line. It is mainly used in scripting.

   echo "Hello, World!"

15. rm

The rm command is used to remove objects such as files, symbolic links and so on from the file system while the rm -r command is used to delete an entire directory.

   rm filename
         rm -r directory 
         rm *.txt

This command removes all files in the current directory with a .txt extension.

16. cp

The cp command is used for copying files from one location to another.

   cp oldfile.txt newfile.txt

17. mv

The mv command moves files and directories from one directory to another or renames a file or directory. If you move a file or directory to a new directory, it retains the base file name.

   mv oldfile.txt newfile.txt

18. useradd

useradd is a Linux command for creating a new user. It requires various options to add and set up an active user account.

   sudo useradd john

22. userdel

The userdel command deletes an existing user. It does not delete the files in the user's home directory unless you supply the -r option. 

   sudo userdel john

23. usermod

usermod command or modify user is a command in Linux that is used to change the properties of a user in Linux through the command line.

   sudo usermod -l newname oldname

24. groupadd

The groupadd command creates a new group account using the values specified on the command line plus the default values from the system. 

   sudo groupadd mygroup

25. groupdel

The groupdel command modifies the system account files, deleting all entries that refer to GROUP. 

   sudo groupdel mygroup

26. su

The su command changes user credentials to those of the root user or to the user specified by the Name parameter

   su username

27. sudo

The sudo command allows you to run programs with the security privileges of another user.

   sudo su

28. which

It is a command which is used to locate the executable file associated with the given command by searching it in the path environment variable. 

   which ls

29. help

The help command provides information on built-in commands. 

   [command] --help

30. man

man command in Linux is used to display the user manual of any command that we can run on the terminal. It provides a detailed view of the command.

   man chmod[command]

31. locate

It is a utility which serves to find files on filesystems. It searches through a pre-built database of files. 

   locate filename

32. find

It can be used to find files and directories and perform subsequent operations on them. It supports searching by file, folder, name, creation date, modification date, owner and permissions.

  find /path/to/search -name filename.txt

33. du

The du (disk usage) command measures the disk space occupied by files or directories. By default, it measures the current directory and all its subdirectories. 

   du -h /path/to/directory

34. uptime

It tells how long the Linux system has been running.

  uptime

35. last

The last command in Linux is used to display the list of all the users logged in and out since the file `/var/log/wtmp` was created.

   last username

36. reboot

reboot command is used to restart or reboot the system. In a Linux system administration, there comes a need to restart the server after the completion of some network and other major updates.

   reboot [OPTIONS]

37. lscpu

It is used to get detailed information about the CPU installed on your system.

  lscpu

38. lsblk

lsblk lists information about all available or the specified block devices. The lsblk command reads the sysfs filesystem and udev db to gather information.

   lsblk [OPTIONS]
